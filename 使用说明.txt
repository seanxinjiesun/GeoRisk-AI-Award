【GeoRisk-AI-Award 使用说明】

一、项目目标

本系统可部署为公网可访问网页（Streamlit Cloud），适合微信分享给评委或同事直接使用。

二、环境要求

* Python 3.10+
* 可联网（AI分析依赖 Claude API）

三、安装依赖

进入项目目录后执行：

pip install -r requirements.txt

四、本地配置与启动

1）推荐使用 Streamlit secrets
在项目目录创建：`.streamlit/secrets.toml`

内容示例：
ANTHROPIC_API_KEY = "your_api_key"
BASE_URL = "https://your-base-url"  # 可选

2）启动系统
python -m streamlit run app.py

3）访问地址
http://localhost:8501

五、核心功能

1. 矿山文件分析（模块1）
   - 支持 PDF / DOCX / TXT / JPG / PNG
   - 可在模块内查询矿山记录并导出Excel

2. 贸易测算决策（模块2）
   - 报价、利润、风险与图表
   - 可在模块内查询贸易记录并导出Excel

3. 指标看板
   - 展示运行次数、成功率、准确率、效率提升等

六、Streamlit Cloud 部署步骤（公网分享）

1）将项目推送到 GitHub
2）登录 Streamlit Cloud，新建应用（New app）
3）选择仓库与分支，入口文件填：app.py
4）在应用 Secrets 中配置：
   ANTHROPIC_API_KEY = "your_api_key"
   BASE_URL = "https://your-base-url"  # 可选
5）点击 Deploy，获得公网链接
6）将链接分享给评委（可微信直接打开）

七、常见排查

* AI调用失败：检查 ANTHROPIC_API_KEY 是否正确
* 使用代理时失败：检查 BASE_URL 是否可达
* 云端偶发失败：重启应用并重新测试样例

八、注意事项

* 前端不会暴露 API_KEY 与 BASE_URL
* 汇率接口异常时系统会自动使用回退值
* 建议评审前准备PDF/PPT结果备份以应对网络波动
